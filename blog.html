Stop Layout Shifts with Native Aspect Ratio

✍️ Ujwal Subedi • 📅 May 27, 2026 • 📍 Shivasatakshi, Nepal

We have all been there: you are reading an article on your phone, you go to tap a link, and—boom!—the page suddenly jumps, causing you to accidentally tap an ad or a completely different button. This frustrating experience is known as Cumulative Layout Shift (CLS), and it is one of the most common user experience issues on the modern web.

Why does this happen? When a browser loads a webpage, it parses the HTML layout first. If your images or videos don’t have defined dimensions, the browser has no idea how much vertical space to reserve. Once the visual media finally finishes downloading, the browser suddenly shifts the rest of your text downward to fit the image.

// Reserve space instantly in one line
img, video {
  width: 100%;
  aspect-ratio: 16 / 9;
  object-fit: cover;
}

In the past, developers had to rely on complex CSS hacks (like the percentage padding workaround) to maintain fluid layouts. Luckily, modern CSS introduces the beautifully simple aspect-ratio property. By using this property, you tell the browser the exact proportions of your container beforehand, reserving the space instantly while the asset loads quietly in the background. Mastering Frosted Glass UI with CSS Backdrop-Filter

✍️ Ujwal Subedi • 📅 May 28, 2026 • 📍 Shivasatakshi, Nepal

Modern user interface design is increasingly moving toward tactile, translucent elements. One of the most sought-after visual aesthetics in web design is **Glassmorphism**—creating elements that mimic physical frosted glass panels. While simple opacities only wash out your colors, authentic glassmorphism refracts and blurs elements passing directly underneath it.

The secret behind this effect is the native CSS property: backdrop-filter. Unlike a standard filter (which blurs the element itself), backdrop-filter applies a mathematical blur only to the visual layers directly below the container.

/* The Modern Frosted Glass Recipe */
.glass-panel {
  background: rgba(255, 255, 255, 0.2);
  backdrop-filter: blur(12px);
  border: 1px solid rgba(255, 255, 255, 0.25);
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.15);
}

To achieve a convincing frosted-glass look, pair a slightly translucent, high-contrast background color with a thin translucent white border. This mimics the light reflecting off the cut physical edge of glass. You can test and toggle the live glass-blur filters down below!
